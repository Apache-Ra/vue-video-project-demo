# vue-video-project-demo
> 基于video.js 播放m3u8流媒体
## 组件和工具类
``` bash
    axios           数据请求
    fastclick       点击延迟
    less
    less-loader     css预加载
    lodash          数组处理
    uuid ID         uuid.v4()
    vuex            状态分发
    Axios           二次封装axios
    LocalStage      LocalStroage封装
    Cookie          Cookie封装

    "video.js": "^7.6.5",
    "videojs-contrib-hls": "^5.15.0",
```
## Build Setup
``` bash
# 下载
npm install
# 运行
npm run dev
# 打包
npm run build
```
